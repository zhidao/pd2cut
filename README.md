# pd2cut ver.2.0
                                                by Zhidao
                                                2004.11.23. 作成
                                                2022. 1. 2. 最終更新
--------------------------------------------------------------------
### 【説明】

工画堂 PowerDoLLS2 plus Dash の*.pacファイルをBMPファイルに分割する
ツール。

--------------------------------------------------------------------
### 【コンパイル】

makeすれば実行ファイルpd2cutが出来ます。

--------------------------------------------------------------------
### 【使い方】

% pd2cut <.pac file1> <.pac file2> ...
とすれば、それぞれの.pacファイルの中身がカレントディレクトリに切り出
されます。ファイル名には4桁の通し番号（開始番号0000）が付きます。
